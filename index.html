Micro-Active

Interactive Web Components for Microscopy Education

About

Interactive web components designed to help students understand concepts in microscopy. Each component is self-contained and can be embedded in Moodle lessons or used in any web page.

Built with: Lit.js, Chart.js, and Shoelace

Airy Pair

Two overlapping Airy disk patterns with adjustable NA, wavelength, pixelation, and noise. Includes Rayleigh, Abbe, and Sparrow resolution criteria.

PSF Resolution Airy Disk

Diffraction & Focusing

Plane wave diffraction through single or double slits, or focusing through a lens aperture. Huygens-Fresnel wavelet summation with three visualization modes.

Diffraction Huygens Lens / PSF

Usage in Moodle

To embed any component in a Moodle lesson:

  1. Upload the component's .js file to your Moodle files
  2. In your lesson, add an HTML block
  3. Insert the following code: 
    <script type="module" src="/path/to/component.js"></script>
<component-name></component-name>
