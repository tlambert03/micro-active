Interactive Web Components for Microscopy Education

About

This collection contains interactive web components designed to help students understand concepts in microscopy. Each component is self-contained and can be embedded in Moodle lessons or used in any web page.

Built with: Lit.js, Chart.js, and Shoelace (Web Awesome)

Available Components

Gaussian Profile

Interactive visualization of a Gaussian distribution with adjustable width. Demonstrates the point spread function (PSF) concept in optical microscopy.

PSF Resolution Optics

Usage in Moodle

To embed any component in a Moodle lesson:

  1. Upload the component's .js file to your Moodle files
  2. In your lesson, add an HTML block
  3. Insert the following code: 
    <script type="module" src="/path/to/component.js"></script>
<component-name></component-name>
