Diffraction Through Slits

Interactive Huygens-Fresnel wavelet simulation

About This Simulation

This component visualizes diffraction of a plane wave passing through one or two narrow slits, computed using the Huygens-Fresnel principle. Every point within each slit opening acts as a source of a secondary cylindrical wavelet. The observed pattern is the coherent superposition of all these wavelets.

U(x, y) = ∑ (dy / r) · exp(ikr) / √r · Δx'

k = 2π/λ — wavenumber
r = distance from each source point to the observation point
dy/r — obliquity factor (suppresses backward radiation)

Display Modes

Wave: Shows the instantaneous real part of the field as grayscale (white = positive peak, black = negative trough, gray = zero).

Wavelets: Highlights individual Huygens wavelets as concentric arcs emanating from selected source points within the slit(s).

Wavefronts: Draws only the crests of the wave as bright traveling lines, fading where amplitude is low.

Key Physics

Single slit: The central maximum widens as slit width decreases. First minimum at sinθ = λ/a.

Double slit: Rapid cos² interference fringes are modulated by the single-slit diffraction envelope. Fringe spacing is λL/d where L is the screen distance and d is slit separation.
